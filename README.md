# Payroll Management System using SQL

## Project Overview
This project is an advanced SQL-based Payroll Management System designed to manage employee records, attendance, and payroll details.

## Features
- Employee Management
- Attendance Tracking
- Payroll Management
- Salary Analysis
- Trigger for Automatic Net Salary Calculation
- Stored Procedure for Bonus Updates

## SQL Concepts Used
- Joins
- Aggregate Functions
- Views
- Stored Procedures
- Triggers
- CTE
- Subqueries
- CASE Statements
- Window Functions

## Tables Used
- Employees
- Attendance
- Payroll

## Tools Used
- MySQL
- SQL Workbench
- GitHub
